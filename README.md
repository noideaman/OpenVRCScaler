# OpenVRCScaler

*Yes it's vibe coded, but I've reviewed and tested it. This is free, please don't complaint about AI usage.*

OpenVRCScaler is a Python application that enables OSC-based avatar scaling in VRChat. It reads scaling parameters from a custom scaling menu, calculates your target height, and sends time-interpolated scaling updates back to VRChat over OSC.

## Requirements

* Python 3.7 or higher
* `python-osc`

## Installation and Execution

1. Install the required dependencies:
   ```bash
   pip install -r requirements.txt

2. Run OpenVrcScaler:
    ```bash
    python open_vrc_scaler.py 

## Avatar Installation

1. Download the latest version of `OpenVRCScaler.unitypackage`.
2. Import the package into your Unity avatar project.
3. Drag and drop the OpenVRCScaler prefab onto your avatar root.
4. Upload you avatar.

OpenVRCScaler.exe **must** be running for the new scaling options to work.
